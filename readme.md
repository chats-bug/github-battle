## Getting Started
Follow the steps listed below to set up the project in your local machine.


### Prerequisites

* Install [Node.js](https://nodejs.org/en/download/)


### Installation
To get a local copy up and running follow these simple steps.
1. Clone the repo
	```sh
	git clone
	```
2. Install NPM packages
	```sh
	npm install
	```
3. Run the app
	```sh
	npm run dev
	```
